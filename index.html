A Virtual Gridlock

To help districts that can’t handle the increased local toll, the state has instituted a program called recapture, that redistributes funds from property rich districts to property poor districts.

While the program seems well intentioned to improve equity in school fundings, in practice both rich and poor districts find themselves stuck in undesirable situations. Let’s take a look...
